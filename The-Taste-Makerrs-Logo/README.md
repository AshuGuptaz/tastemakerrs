# The Taste Makerrs — Logo

An elegant two-tier celebration cake with a single lit candle — a nod to *"a cake for your favourite memories."* Set with the **Prata** wordmark (a warm high-contrast serif) and a small terracotta full-stop.

## Files
- **/svg** — master vector artwork (use for print, signage, web, anything that scales).
- **/png** — high-resolution transparent PNGs.
- **/favicon** — website + app icons (`app-icon.svg`, `favicon-32/48/64/180/192/512.png`, transparent `mark.svg`).

Every logo comes in 4 colourways: **full** (espresso + terracotta), **espresso** (mono), **terracotta**, and **reversed** (for dark backgrounds).
Forms: **primary** (stacked), **horizontal**, **wordmark**, and the **mark** in line + solid.

## Use
- Primary = the stacked lockup. Use the **line** mark on its own for large/elegant uses; the **solid** mark for small sizes, favicons and stamps.
- Give it clear space; keep terracotta as the only accent; lead with cream.

**Colours** — Espresso `#241A15` · Terracotta `#D67A47` · Cream `#FBF7F0`
**Type** — Prata (wordmark) · Marcellus (tagline caps)
